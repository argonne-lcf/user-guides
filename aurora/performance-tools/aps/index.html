    Application Performance Snapshot¶

    Introduction¶

    Use Application Performance Snapshot for a quick scan of performance aspects that relate to compute-intensive applications:

    • MPI usage
    • OpenMP usage
    • IntelÂ® oneAPI Collective Communications Library (oneCCL) usage
    • CPU/GPU usage
    • Efficiency of memory access
    • Vectorization
    • I/O and memory footprint

    Application Performance Snapshot displays key optimization areas and suggests specialized tools for tuning particular performance aspects, such as Intel VTune Profiler and IntelÂ® Advisor. You can run Application Performance Snapshot on large MPI workloads to analyze scalability issues.

    Instruction¶

    Loading the module¶

    $ module load oneapi
$ aps --version
Intel(R) VTune(TM) Profiler 2025.0.1 (build 629235) Command Line Tool
Copyright (C) 2009 Intel Corporation. All rights reserved.

    Running with an MPI application¶

    Collection on compute nodes¶

    $ mpirun <mpi arguments> aps -r <aps_result_dir> <application> <args>

    HTML report generation¶

    $ aps --report <aps_result_dir>
